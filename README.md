# vue3-crud

## 项目设置
```
npm install
```

### 编译和热重载用于开发
```
npm run serve
```

### 在生产环境中编译和压缩 
```
npm run build
```

### 自定义配置
See [Configuration Reference](https://cli.vuejs.org/config/).



> 本项目主要用于简单的增删改查，涉及到的有分页查询、模糊查询、联表查询、多表之间的更新删除
